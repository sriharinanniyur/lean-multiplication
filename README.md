# Lean proofs of Karatsuba and Toom-Cook multiplication

Lean 4 correctness proofs of the Karatsuba and Toom-Cook algorithms, assisted by Aristotle.

This repository is an archive of our early work; we have covered several more multiplication algorithms since. See [here](github.com/sriharinanniyur/fast_verified_mul).

Algorithms covered:
- Karatsuba's algorithm
- Toom-Cook algorithm with integer evaluation points
